## Sistema de Cadastro de Usuários
Este programa em Java simula um sistema simples de cadastro de usuários, permitindo que o usuário:
Cadastrar um novo usuário, buscar um usuário pelo email, remover um usuário e listar todos os usuários cadastrados.

## Tecnologias
Linguagem: Java, Bibliotecas: Scanner (Para ler a entrada de dados) e Classes: User, UserServices (Para manipulação dos dados dos usuários)
